Recent projects

Hajimaru

At first, Hajimaru was built as a way to design my own set of startup and bookmarks manager to replace the ones that comes by default with any internet browsers. It has since evolved to have a templating support for easy modification of aliases and bookmarks without having to touch the source code. A live demo is available on the repository of the project.

July 2016
Website: github.com/finiks/hajimaru

Bourgeois.io

This very site was built from the ground up using the Materialize framework along with my own set of HTML5 and CSS3 files. It was built with the idea of having a one-page website where I could host a brief summary of what I do. The background was created using particles.js and my own custom configuration and some work was put in place to make sure that the entire page was fully responsive regardless of the device it was viewed on.

February 2016
Website: bourgeois.io

Apollo

Knowing that there was no easy way to manage my own backlog of media titles such as movies and video games, I decided to create Apollo. Built from scratch with the Golang language, it uses a terminal emulator as a front-end to interactively display my entire backlog. Once a new title is added, Apollo will then fetch up-to-date metadata and append it to that entry. The entries can then be searched for, rated, and edited.

Summer 2015
Website: github.com/finiks/apollo
